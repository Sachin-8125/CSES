# CSES Solutions

A comprehensive and well-organized archive of **accepted solutions** to the [CSES Problem Set](https://cses.fi/problemset/), implemented in **C++**.

This repository is intended for learning and revision of core data structures and algorithms, with solutions spanning from introductory problems to advanced topics.

---

## 📌 About

The CSES Problem Set is a popular platform for mastering competitive programming fundamentals.  
This repository collects accepted solutions and keeps them grouped by topic for easier navigation.

### Goals of this repository

- Maintain clean and correct C++ implementations.
- Provide topic-wise organization for structured practice.
- Serve as a quick reference during revision.
- Track long-term progress across the full CSES set.

---

## 🧠 Topics Covered

Solutions in this repository correspond to standard CSES categories such as:

- Introductory Problems
- Sorting and Searching
- Dynamic Programming
- Graph Algorithms
- Range Queries
- Tree Algorithms
- Mathematics
- String Algorithms
- Geometry
- Advanced Techniques

> As the repository grows, additional solutions and refinements are added.

---

## 🛠️ Language

- **C++ (100%)**

The solutions are written with a competitive programming style focused on correctness and efficiency.

---

## 🚀 How to Use

1. Open the relevant topic folder.
2. Pick the CSES problem you want to study.
3. Read the corresponding solution file.
4. Compile and run locally if needed:

```bash
g++ -std=c++17 -O2 -Wall -Wextra -o solution solution.cpp
./solution
```
